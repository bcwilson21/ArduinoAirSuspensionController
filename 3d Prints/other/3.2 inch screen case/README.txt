Case designed by Cristhian at Ruin Air https://www.instagram.com/ruin_air/ 

This is designed for 3.2" integrated esp32 screen ESP32-2432S032C https://www.amazon.com/dp/B0CLGDHS16

Short case is for no battery
thick case is for use with a battery

You can use the split parts feature in your slice to split it into components and delete the parts you don't want to print

If you want a carbon fiber texture on the case you can use a printing sheet like this https://www.amazon.com/dp/B0CY7VKJ58
